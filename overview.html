Overview

The report begins with background information about Notion and Obsidian. It includes an in-depth look at the user flow during onboarding and basic note-taking functions of each app, the methodology for user testing, and the elements of UX design. An analysis of the methodology for testing and data provides insight into effectiveness of the test environment. Finally, recommendations for an app prototype completes the process.
